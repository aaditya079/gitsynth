## 🌟 Introduction

**GitSynth** is a high-fidelity, client-side web application designed to help developers create exceptionally gorgeous, stats-heavy, and dynamic GitHub Profile READMEs. Inspired by stats-first layouts, this tool provides a premium real-time customizer and simulated markdown visualizer in one unified dashboard.

---

## ✨ Features

- 🌌 **Premium Dark Glassmorphic Theme:** Curated harmonious color palettes with vibrant neon-cyan (`#00F7FF`) accents and smooth glow animations.
- ⚡ **Real-Time Interactive Customizer:** Watch your markdown visual structure update dynamically as you type.
- 💻 **Personalized Tech Badges:** Select from a reactive, curated catalog of shields.io badges for programming languages, backend frameworks, databases, and tools.
- 🎯 **Roadmap and Bio Bullet Builders:** Easily compile your B.Tech student milestones, learning targets, active projects, and annual roadmap items.
- 📋 **Seamless Export Options:** One-click copy-to-clipboard action and instant direct `README.md` download.
- 🌐 **Zero Dependencies:** Built purely with Vanilla HTML, responsive CSS Grid/Flexbox, and lightweight client-side JavaScript.

## 💻 Running Locally

To run the customizer application locally on your computer:

1. Clone this repository:
   ```bash
   git clone https://github.com/aaditya079/gitsynth.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gitsynth
   ```
3. Start a local server:
   ```bash
   # Using Python
   python -m http.server 8080
   ```
4. Open your browser and go to: **`http://localhost:8080`**
